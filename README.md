☢️ RadiationGPT

AI Assistant for **Nuclear Physics & Radiation Measurement**

RadiationGPT is an AI-powered assistant designed to help understand **radiation science, nuclear physics concepts, and radiation measurement calculations**. The application combines **Artificial Intelligence with radiation measurement tools** to create an interactive learning and analysis platform.

---

## 🚀 Features

• AI Chat Assistant for Nuclear Physics questions  
• Radiation Detector Knowledge Module  
• Minimum Detectable Activity (MDA) Calculator  
• Radioactivity Activity Calculation Tool  
• Radioactive Decay Correction Calculator  
• Interactive AI dashboard for radiation science

## 📊 Radiation Calculators

The application includes practical radiation measurement tools:

1️⃣ Minimum Detectable Activity (MDA)

\[
MDA = \frac{2.71 + 4.65\sqrt{B}}{E \times T}
\]

Where  
B = Background counts  
E = Detector efficiency  
T = Counting time  

2️⃣ Activity Calculation

\[
Activity = \frac{C - B}{E \times T}
\]

3️⃣ Radioactive Decay Correction

\[
A = A_0 e^{-0.693t/T_{1/2}}
\]
